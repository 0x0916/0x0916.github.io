数据结构

hlist_head和hlist_node用于散列表，分别表示列表头（数组中的一项）和列表头所在双向链表中的某项，两者结构如下:

include/linux/types.h(line 190) 
struct hlist_head {
	struct hlist_node *first;
};
include/linux/types.h(line 194) 
struct hlist_node {
	struct hlist_node *next, **pprev;

Cgroup之cpuacct子系统

2018-05-20
kernel cgroup
2754 words 6 mins read

cpuacct子系统（CPU accounting）会自动生成报告来显示cgroup中任务所使用的CPU资源，其中包括子群组任务。报告有两大类：

  • usage: 统计cgroup中进程使用CPU的时间，单位为纳秒。
  • stat: 统计cgroup中进程使用CPU的时间，单位为USER_HZ。
